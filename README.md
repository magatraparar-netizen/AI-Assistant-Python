# AI Assistant Python

## 📌 Description
This is a Python AI Assistant built using the Google Gemini API.

## ✨ Features
- AI Chat Assistant
- Live Google Search
- Credit System
- Chat History Saving
- Error Handling

## 🛠 Technologies Used
- Python
- Google Gemini API

## ▶️ How to Run

1. Install the required package:
```bash
pip install google-genai
```

2. Run the project:
```bash
python main.py
```
